# 🏡 Arielove Kolibe — Zvanični Sajt

Zvanični sajt za **Arielove Kolibe** — autentične planinske brvnare u srcu Mokre Gore, u neposrednoj blizini Šarganske osmice i Drvengrada.

---

## 📁 Struktura Projekta

```
arielove kolibe apartmani/
│
├── index.html              # Glavna stranica sajta
├── style.css               # Svi stilovi (CSS)
├── script.js               # Interaktivnost (lightbox, navigacija)
├── README.md               # Ovaj fajl
│
└── assets/
    └── images/
        ├── cabin1/         # Fotografije brvnare sa 1 sobom (1.jpg – 8.jpg)
        └── cabin2/         # Fotografije brvnare sa 3 sobe (21.jpg – 28.jpg)
```

## 🚀 Postavljanje na Internet (Deployment)

### Opcija 1: GitHub Pages (besplatno, preporučeno)

1. Otvorite [github.com](https://github.com) i napravite nalog (ako ga nemate)
2. Kliknite **"New repository"**, nazovite ga npr. `arielove-kolibe`
3. Otpremite sve fajlove iz ovog foldera (uključujući `assets/` folder sa slikama)
4. Idite na **Settings → Pages**
5. Pod **"Branch"** izaberite `main` i kliknite **Save**
6. Sajt će biti dostupan na: `https://vase-korisnicko-ime.github.io/arielove-kolibe`

### Opcija 2: Netlify (besplatno, drag & drop)

1. Otvorite [netlify.com](https://netlify.com) i napravite nalog
2. Na Dashboard-u kliknite **"Add new site → Deploy manually"**
3. Prevucite ceo folder projekta u označenu zonu
4. Netlify automatski generiše link, npr. `https://arielove-kolibe.netlify.app`
5. Opciono: povežite sopstveni domen (npr. `arielovekolibe.rs`)

### Opcija 3: Hostinger / cPanel Hosting (plaćeni, sa domenom)

1. Kupite domen i hosting paket (npr. `arielovekolibe.rs`)
2. Prijavite se na cPanel / File Manager
3. Otvorite folder `public_html`
4. Otpremite sve fajlove projekta (`index.html`, `style.css`, `script.js`, `assets/`)
5. Sajt je odmah dostupan na vašem domenu
